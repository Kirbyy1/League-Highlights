# League Highlights v25 — Better Game Metadata + Match Timeline

## Added
- Final K/D/A saved for every completed game.
- Actual match duration, game mode, map, result, team and start time shown in the library.
- Normalized Victory / Defeat labels.
- Riot in-game event timestamps shown as a visual match timeline.
- Clickable timeline markers open the corresponding clip in the built-in player.
- Timeline markers emphasize high-score plays, pentakills and objective steals.
- Clips in a game are ordered by their in-game event time.

## Compatibility
Existing clips remain visible. Older clips that do not contain the new metadata show only the information already available. New games automatically receive the richer metadata when the match finishes.
